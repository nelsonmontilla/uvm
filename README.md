# uvm
Para la ejecucion de este repositorio hay que seguir los siguientes pasos:

1:Contar con VSC en tal de no contar con dicho progrma, se puede descargar del siguiente enlace : https://code.visualstudio.com/download
2: Contar con la Extension "Live Server" este se usa para ver los cambios en vivos que hagamos en el archivo.
3: Descomprimir los archivos y almacenarlos en un carpeta 
4:importar la carpeta a VSC y Inicar el Live Server 
5: Una vez ejecutado el LV , podremos ver la pagina por unos de los puertos disponibles (http://127.0.0.1:5501)
